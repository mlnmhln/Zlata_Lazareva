# Мишкин день

Статический сайт книги «Мишкин день».

## Структура

- `index.html` — основной сайт.
- `book-animation.html` — встроенная HTML-анимация книги.
- `assets/fonts` — локальные шрифты.
- `assets/images` — изображения сайта.

## Запуск

Откройте `index.html` в браузере или включите GitHub Pages для репозитория.
